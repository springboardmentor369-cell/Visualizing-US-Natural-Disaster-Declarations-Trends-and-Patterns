### Project Description

This project focuses on analyzing U.S. natural disaster declaration data using Power BI. The objective was to identify trends, patterns, and insights related to disaster occurrences across different states and years.

### Tools & Technologies Used
- Power BI
- FEMA Dataset
- Data Cleaning & Transformation
- DAX Measures
- Data Modeling (Star Schema)

### Key Features
- Interactive dashboard with slicers
- Year-wise disaster trend analysis
- State-wise comparison
- KPI cards for disaster count
- Clean and structured data model

### Key Insights
- Identified states with highest disaster frequency
- Observed trend patterns over years
- Compared disaster types across regions
- Highlighted peak disaster periods
